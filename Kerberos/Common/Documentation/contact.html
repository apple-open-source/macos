Kerberos for Macintosh Contact Information

For all problems, questions, and suggestions regarding Kerberos on Mac OS X, you should contact Apple Computer. They will relay questions to MIT as appropriate. See the support page for more information on how to contact Apple.

If you have general Kerberos questions or just want to become more involved in the Kerberos community, MIT runs several mailing lists for discussing Kerberos. Please visit the MIT Kerberos mailing list page for more information.

If you want to send us comments or questions about the information on these web pages, please send mail to the MIT Kerberos Development Team: krbcore@mit.edu.
