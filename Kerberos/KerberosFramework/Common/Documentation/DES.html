DES in KfM

This documentation only applies to Mac OS X 10.2, 10.3 and 10.4.

Kerberos for Macintosh includes an implementation of the Data Encryption Standard (DES) protocol.

It exports the DES functions from the Kerberos v4 implementation.

You can see which DES functions are exported in Kerberos for Macintosh by consulting /System/Library/Frameworks/Kerberos.framework/Headers/des.h.

The latest release is included with Mac OS X 10.2 and later.
