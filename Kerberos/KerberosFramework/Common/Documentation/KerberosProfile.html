Kerberos Profile Library in KfM

This documentation only applies to Mac OS X 10.2 and 10.3.

The Kerberos Profile Library is part of the Kerberos v5 library in Kerberos for Macintosh. It provides interfaces for reading and writing the Kerberos configuration in the edu.mit.Kerberos file.

Please refer to profile.h in the /System/Library/Frameworks/Kerberos.framework/Headers directory.

The latest release is included with Mac OS X 10.2 and later.
