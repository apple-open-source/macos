Credentials Cache API

The Credentials Cache API provides an API for accessing the in-memory credentials cache for Kerberos v4 and Kerberos v5 tickets. The Credentials Cache API is used by other libraries in the Kerberos framework and should not normally be needed by Kerberos applications.

API Documentation:

  • Credentials Cache API version 3
  • Credentials Cache API version 2

Please read the Credentials Cache library Bugs List and Version History for information on the latest release of the Credentials Cache library.
