Dbc

 

#include <db_cxx.h>


class Dbc { ... };

Description

The Dbc object is the handle for a cursor into a Berkeley DB database. The handle is not free-threaded, and cursors may not span threads; nor may cursors be used by more than a single thread. If the cursor is to be used to perform operations on behalf of a transaction, the cursor must be opened and closed within the context of that single transaction. Once Dbc::close has been called, the handle may not be accessed again, regardless of the method's return.
