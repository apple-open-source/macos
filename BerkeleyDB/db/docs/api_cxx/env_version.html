DbEnv::version

 

#include <db_cxx.h>


static char *
DbEnv::version(int *major, int *minor, int *patch);

Description

The DbEnv::version method returns a pointer to a string containing Berkeley DB version information. If major is non-NULL, the major version of the Berkeley DB release is stored in the memory to which it refers. If minor is non-NULL, the minor version of the Berkeley DB release is stored in the memory to which it refers. If patch is non-NULL, the patch version of the Berkeley DB release is stored in the memory to which it refers.

Class

DbEnv
